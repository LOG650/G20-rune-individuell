## Hva som er viktigst å få avklart

### Kritisk bare for normativ nasjonal modell eller rangering

Disse punktene må være avklart dersom rapporten skal konkludere om kapasitet eller bemanningsriktighet ved andre enkelt-sentraler. I denne rapporten kan de håndteres som forbehold, siden nasjonal del brukes deskriptivt.

1. **Bemanning som inngår i c_eff**
   - Er MOB-tallet minimum, normalbemanning eller planlagt bemanning?
   - Er vaktleder inkludert i tallet?
   - Hva er reell minimumsbemanning på dag/natt/helg?

2. **ABA og service**
   - Registreres ABA-service/test i LEO/BRIS eller utenfor?
   - Håndteres service av 110-operatør eller dedikert personell?
   - Endrer dette seg ved helg, sykdom eller høy belastning?

3. **Store dataavvik**
   - Skyldes avviket reell drift, lokal registreringspraksis eller feil/ulik rapportering?
   - Kan tallene brukes direkte, må de korrigeres, eller må sentralen merkes med forbehold?

### Bør avklares hvis avviket omtales i rapporttekst

- Hva MOB-feltet «mottatte 110-anrop» faktisk inkluderer.
- Om VL normalt besvarer nødanrop.
- Om oppdrag overføres til nabosentral etter ventetid/køgrense.
- Om spesielle lokale forhold i 2025 påvirker tallene.

### Nice to have

- Detaljerte bindingstider per kategori.
- Subjektiv opplevelse av bemanning.
- ROS-/beredskapsanalyse og dimensjoneringsgrunnlag.
- Vikarandel/bakvakt.

## Prioritert oppfølging

Oppfølging bør begrenses til avvik som faktisk kan påvirke rapportens tolkning. Øvrige manglende svar merkes som forbehold.

1. **Tromsø 110** - kun hvis tabellen med oppdrag/c_eff skal vektlegges, fordi lav MOB-bemanning gir stor c_eff-usikkerhet.
2. **Oslo 110 / Sør-Øst 110** - hvis ABA-forskjeller omtales tydelig, fordi 0 % L-aba trolig skyldes registreringspraksis.
3. **Midt-Norge 110** - bekreft serviceavklaring for 2025 hvis Midt-Norge brukes som eksempel på funksjonsdifferensiering.
4. **Møre og Romsdal / Nordland / Agder / Øst** - bare ved omtale av DSB/MOB-avvik, skjult-rate eller alarmbehandlingstid.
5. **Sør-Vest 110** - intern avklaring bør lukkes fordi dette er primærcase.
6. **Vest 110 / Innlandet 110 / Finnmark 110** - eksisterende svar er tilstrekkelige for deskriptiv bruk; åpne punkt merkes med forbehold.
